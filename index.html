Дипломный проект Дмитриева Андрея | Специальность 40.02.03 Юриспруденция

О проекте

Реализация требований 152-ФЗ: отзыв согласия, удаление данных, экспорт, логирование, авторизация, безопасность, контейнеризация.

Стек технологий

  • Node.js + Express
  • MySQL 8.0
  • Nginx (reverse proxy, HTTPS, rate limiting)
  • Docker + Docker Compose
  • phpMyAdmin, HTML, CSS, JavaScript
